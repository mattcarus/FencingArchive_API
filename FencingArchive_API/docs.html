FencingArchive.net API

/

/twitter/status

Latest Twitter status messages

Example: /twitter/status

/twitter/profile

@backonthepiste Twitter profile

Example: /twitter/profile

/fencer/:id

Get the fencer object with id :id

Example: /fencer/123

/competition/:id

Get the competition object with id :id

Example: /competition/123

/club/:id

Get the club object with id :id

Example: /club/123

/series/:id

Get the series object with id :id

Example: /series/123

/query/:queryName/:options

Run the query named :queryName with the option :option

  • lastNCompetitionLinks - Return links to the last n competitions. Option value is the number of links to return
  • recentSeries - return the most recent series

Example: /lastNCompetitionLinks/3

/search/term

Passthrough node to elastic search instance at http://api.fencingarchive.net:9200/. Note that these results are currently returned as JSON (most other things are XML).

Example: /search/bob
